# FractumTria
## Controls:
- Dreta: Fletxa dreta / D
- Esquerra: Fletxa esquerra / A
- Salt: Fletxa amunt / W
- Canvi de color: Z / K
- Disparar: X / J / Botó esquerra ratolí

## Descarrega i executa:
### Windows:
1. Descarrega i descomprimeix el .zip
2. Executa el .exe

### MacOS:
1. Descarrega i descomprimeix el .zip
2. Obre el .dmg
3. Executa el .app
4. Si surt un avís de seguretat, vés a configuració > seguretat i clica "Obre igualment"
